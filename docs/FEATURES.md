# 功能特性

## 核心能力

### 输入模拟

通过 Better-xCloud 的 inputChannel 发送手柄输入，支持：
- 按键：A/B/X/Y, D-Pad, LB/RB, LT/RT
- 摇杆：左右摇杆 X/Y 轴
- 快捷方法：`tap()`, `hold()`, `combo()`

### 画面识别

基于 OpenCV.js 在 Web Worker 中运行模板匹配：
- 支持 ROI 区域限制
- 降采样加速
- 多尺度自适应

### 任务调度

继承 `BaseTask` 实现自定义任务，支持：
- 独占/后台模式
- 资产热加载
- 配置热更新

---

## 宏录制

记录并回放用户的手柄操作序列。

**基本操作**：
- 新建录制 → 操作 → 停止并保存
- 选择宏 → 播放/暂停/停止/重放

**进阶功能**：
- 批量管理：多选、导出、复制、删除
- JSON 文件导入导出
- 增量录制，仅保存状态变化

---

## 自带任务

| 任务 | 说明 |
|------|------|
| 自动拾取 | 检测拾取提示，自动按 F |
| 自动剧情 | 跳过对话、选项、过场 |

---

## UI 面板

四个 Tab：
- **通用**：匹配阈值、ROI 配置
- **任务**：任务启停、资产预览
- **录制**：宏管理、播放控制
- **系统**：配置导入导出、诊断
